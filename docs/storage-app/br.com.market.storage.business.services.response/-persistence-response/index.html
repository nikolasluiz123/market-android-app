PersistenceResponse

data class PersistenceResponse(var idLocal: Long? = null, var idRemote: Long? = null, var code: Int, var success: Boolean = false, var error: String? = null) : IMarketServiceResponse

Classe que representa uma resposta de persistência, por ser utilizada, por exemplo, como retorno de uma função que salva ou atualiza uma entidade.

Ela tem como objetivo conter as chaves primárias de uma entidade para que, se for necessário, possamos fazer operações com esses dados, por exemplo, realizar alguma busca ou outra operação que necessite do ID.

Função que pode ser utilizada para envolver o bloco de código em um tratamento padrão de exceções que podem ocorrer ao tentar estabelecer uma conexão com o serviço.

A função permite que sejam adicionados novos tratamentos específicos de uma rotina, esses tratamentos específicos devem seguir um padrão:

Para realizar o tratamento, utilize o when da mesma forma como é feito nessa implementação, além disso, seus tratamentos deverão sempre terminar tratando uma Exception, pois, da forma que é feito atualmente está localizada no else do when, e é onde as Exceptions cairiam pro padrão.

Essa função é preparada para o cenário de Autenticação, note que ela obriga o retorno de uma AuthenticationResponse, portanto, utilize-a nesses cenários.
